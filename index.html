🎯 Project Objectives

  • Understand how phishing attacks are constructed
  • Simulate a full campaign in a controlled environment
  • Analyse user susceptibility through metrics
  • Promote cybersecurity awareness

⚖️ Ethical Statement

This project was conducted strictly for academic purposes within an isolated lab environment. No real user data was collected. All demonstrations used simulated targets only. Using GoPhish against real individuals without consent is illegal and unethical.

🛡️ How to Stay Safe

  • Always check the full URL before entering credentials
  • Never click links in unsolicited emails
  • Enable two-factor authentication on all accounts
  • Report suspicious emails to your IT team
  • If unsure, navigate directly to the website
